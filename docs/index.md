# index

## 浏览器支持

IE10+


## 命名

### 前缀

`ui-`

e.g
```css
.ui-button{}
```

### 扩展

`[缩写]-[扩展]`

e.g
```css
.ui-button {} // 正常按钮
.btn-primary {} // 焦点
.btn-warn {} // 警告
```

**常用**

> 边写边补充中


导航 | nav | nav
文档 | doc | doc
菜单 | menu | menu
选项卡 | tab | tab
按钮 | button | btn
错误 | error | err
图片 | image | img
标题 | title | tit
注册 | regist | reg
输入 | input | ipt


### 尺寸

`large`
`small`


### 状态

`disabled` 禁用
`show` 显示
`hide` 隐藏
`error` 错误
